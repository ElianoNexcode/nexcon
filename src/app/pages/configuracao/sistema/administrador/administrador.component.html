Configuração do Administrador

Login de Administrador

O Administrador terá acesso a todos os recursos do Sistema, sem nenhuma restrição.

Somente o Administrador terá acesso a esta tela do Sistema.

Recomendamos criar um nível de operação chamado Administrador com todos os privilégios de acesso ao sistema caso seja necessário o cadastramento de outros Administradores.

Cuide para que nenhuma pessoa indevida saiba a senha de Administrador do Sistema.
