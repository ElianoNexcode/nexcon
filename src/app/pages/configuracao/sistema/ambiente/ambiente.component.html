Atenção!

Além das imagens salvas em banco de dados, o Sistema utiliza de uma estrutura de diretório para armazenamento em arquivos criptografados. Sendo assim, é necessário dar permissão total ao diretório configurado para que o Sistema possa executar suas operações, como: criar subdiretórios, salvar, ler e excluir arquivos.

É importante também definir um processo de backup deste diretório de imagens assim como o backup do banco de dados.
